# Full Stack User Management System

A full-stack User Management System built using Next.js, Express.js, and MySQL. This application allows users to perform CRUD (Create, Read, Update, Delete) operations with a modern and responsive UI.

---

## Features

* Add new users
* View all users
* Update existing users
* Delete users
* REST API integration
* Responsive modern UI using Tailwind CSS
* Full-stack architecture

---

## Tech Stack

### Frontend

* Next.js
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Installation and Setup

### 1. Clone the repository

https://github.com/Praveen793s/mySQL-CRUD-APP
```

---

### 2. Backend Setup

```
cd backend
npm install
npm run dev
```

Backend will run on:

```
http://localhost:5000
```

---

### 3. Frontend Setup

```
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:3000
```

## API Endpoints

| Method | Endpoint       | Description     |
| ------ | -------------- | --------------- |
| GET    | /api/users     | Get all users   |
| POST   | /api/users     | Create new user |
| PUT    | /api/users/:id | Update user     |
| DELETE | /api/users/:id | Delete user     |
